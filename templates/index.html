Welcome to PeerPrep

A smarter way to manage open-source collaboration

PeerPrep is a web-based Project Admin dashboard designed to simplify the management of open source projects. It provides project admins with tools to efficiently track contributors, manage issues and pull requests, and share announcements with the community. Built using Django, HTML, CSS, and JavaScript, PeerPrep aims to improve collaboration and streamline project workflows for maintainers and contributors alike.
